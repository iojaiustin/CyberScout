# CyberScout
## Your personal threat intelligence assistant

CyberScout is a Cyber-Threat Intelligence bot that verifies anyone's vulnerability to the dangers of the internet. Find if you have any exposed email addresses, passwords, usernames or bank details both in DarkWeb and Clearnet.
